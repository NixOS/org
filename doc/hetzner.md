# Hetzner

We have multiple Hetzner Robot and Hetzner Cloud accounts/projects.

Both [@infra-build](https://github.com/orgs/NixOS/teams/infra-build) and
[@foundation](https://github.com/orgs/NixOS/teams/foundation) members have full
access to the Hetzner account.

## Hetzner Cloud

In the Hetzner Cloud personal Hetzner account holders can be invited into a
[project](https://docs.hetzner.com/cloud/general/faq/#what-are-projects-and-how-
can-i-use-them) to grant them access to manage its resources.
